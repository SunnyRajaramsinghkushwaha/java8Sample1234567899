JAVA8

Java 8 is a revolutionary release of the world's #1 development platform. It includes a huge upgrade to the Java programming model and a coordinated evolution of the JVM, Java language, and libraries

Features Of Java 8


1.Lambda expression −: It adds functional processing capability to Java and make program portable and shoter.
2.New tools −: java 8 new compiler tools and utilities are added like ‘jdeps’ to figure out dependencies.
3.Default method −:It provide interface to have default method implementation.
4.Method references −: Using functions as parameter, Method references Referencing functions by their names                                           instead of invoking them directly
5.Stream API −: New stream API to facilitate pipeline processing.
